## Requirements
To run the web app locally, you will need the following.
- category-encoders
- dvc
- joblib
- matplotlib
- numpy
- pandas 
- path
- pickleshare
- scikit-learn
- scipy
- seaborn
- statsmodels
- xbboost

## Installation
```
git clone https://github.com/ldebele/Employee-Attrition-Prediction
cd Employee-Attrition-Prediction 
make install
```
## Run 
To run the web locally
```
make run
```
